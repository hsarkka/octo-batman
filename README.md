octo-batman
===========

Some front-end technology prototyping

Requirements
------------

* Node.js
* Express
* Bower
* Gulp
* Compass

Quick start:

	npm install -g express bower gulp gulp-concat gulp-uglify gulp-minify-css
	
	npm install

	bower install
	
	gem update --system
	gem install compass
	
	gulp

	gulp watch
	
	node app
